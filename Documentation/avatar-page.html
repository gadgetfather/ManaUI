Avatar

This component consists of user profile pictures, usually found in circular shapes.

Profile Avatar

Profile Avatars can be created using the class 'avatar'. avatar-circle is to make avatar circular.

Sizes Avatar

Size of the avatars can be changed by adding avatar-large | avatar-medium | avatar-small class to the existing avatar class..

Square Avatar

Adding a class 'avatar' to existing class, will generate a square shaped avatar.
