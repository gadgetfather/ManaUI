Introduction to MANA

Mana is an open source styling framework, which helps Developers to use best styles and components, by using pre-defined classnames. Mana helps you to quickly build your Dream Project, by focusing much on Functionality, because we take care of your Designs.

Installation Guide

Personal Modification

Despite the fact that MANA targets fulfilling it's client's need, however much as could reasonably be expected, There are times, when the pre-characterized styles are not adequate. To abrogate Mana's base classes, clients can add their own class to the previous classes, and style them seperately.
